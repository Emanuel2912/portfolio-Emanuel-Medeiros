# Frontend — Emanuel Medeiros Portfolio

Aplicação React do portfólio.

## Scripts

```bash
npm install
npm start
npm run build
```

## Rotas principais

- `/` — portfólio.
- `/curriculo` — currículo online.

O chatbot usa `/api/chat`. Em produção na Vercel, esse endpoint é servido pelo arquivo `api/chat.js` na raiz do projeto.
